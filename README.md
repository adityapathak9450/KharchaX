# 💸 KharchaX

A modern MERN-based finance management platform that helps users track expenses, manage wallets, monitor budgets, and gain spending insights through an intuitive dashboard.

## 🚀 Features

* Expense and income tracking
* Wallet management
* Budget planning and monitoring
* Interactive analytics dashboard
* Real-time financial updates
* Secure authentication and authorization
* Responsive UI for all devices

## 🛠️ Tech Stack

**Frontend**

* React.js
* JavaScript
* CSS / Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**Tools**

* Git
* GitHub
* Postman

## 📦 Installation

```bash
git clone <repository-url>
cd KharchaX
```

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Environment Variables

Create a `.env` file and configure:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

## 🎯 Project Goal

KharchaX is designed to simplify personal finance management by providing users with a seamless way to track spending, manage budgets, and make data-driven financial decisions.
